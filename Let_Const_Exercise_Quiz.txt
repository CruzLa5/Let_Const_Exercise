Let_Const_Exercise_Quiz





1. The difference between var and let is that with var you can reassign and redeclare, but you cannot redeclare with let. With let you can also do block scopes, and not with var. Var can also hoist variables.

2. Var can reassign, redeclare, and hoist variables. With the const key word you cannot reassign or redeclare, and you have to work with block scopes. 

3. The differences between let and const is that with let you can reassign, you cannot reassign with const. 

4. Hoisting is when you you assign variable or a function and it's code is brought to the top of the scope of code. The variable name or function is runs through first as if it were brought to the top of the rest of code.
